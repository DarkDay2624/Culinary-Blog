# Culinary-Blog-Web-App


## Features
- Personal profile.  
- Image upload.  
- Authentication.    
- Blog posts.  
- Comments.  
- Edit blog posts, comments.
- Search posts.  

## Tech Stack
- Node.js and Express.js. Uses server side rendering to render each pages.  
- MongoDB.  
- Cloudinary for image storage and their APIs for image retrieval.  
- Bootstrap.  

## How to run
```
npm install
npm start
```
